Mood Adaptive Chatbot
What is it?
MoodBot is a mood adaptive chatbot that calculates moods from an input statement and generates an response. MoodBot was created as a project for Los Altos Hacks VI.
How it Works
MoodBot works by noticing sentence similarities through cosine similarity. The sentenced are tokenized using NLTK to plug into the cosine formula. We then compared already trained input and output statements agianst each other to find matches that are within a specified threshold. The code in large is in two categories; the mood analysis and chatbot. These two were developed seperately, so you might experience some roughness in the integration of the two functions. In short, the mood analyzer part, like it speaks with its name, analyzes the data in the category of moods - we have categorized the emotion only into two axis, happyness-sadness and anger/suprise-fear. Using this info, the chatbot finds the most similar response that it has been trained with, and returns the value. Due to this, the chatbot is unable to return values for statments that are far off of the data set which it trained on, and is also influenced gravely by the data set it has trained in.
Mood Analyzer
The mood analyzer works easier than you might think. Based off of about 100 initial base sentences(initial statements and responses), the mood analyzer is able to train using input/output data. Here, not only is the emotion for the given statements defined, but the statements compared are also re-defined accordingly to the output created by the training function. The way the output list is formatted is [happyness-sadness, anger/suprise-fear, accuracy], in which happyness is 1, sadness is 0, anger/suprise is 1, fear is 0, and also, the higher the accuracy(integer value), the more it has been trained relative to other outputs. Also with the genInitResponse function, it is also able to grab the response most likely to be used for your input, and return a reply for the given statement.
Training
MoodBot when imported, has no pre-trained data. To train MoodBot, you can use it's training functions to train data from .json files. MoodBot finds conversations within the raw message files by finding time proximity between messages and checking for repeat message authors. After a conversation with one input statement and one output statement is created, MoodBot can find the best matching input statement to an output statement using self.response. Users can use pre-written trainer classes to train parse and format message data from Discord and Google Chat.
